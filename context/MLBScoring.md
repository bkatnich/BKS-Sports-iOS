# DraftKings MLB Classic — Scoring Rules

Source: DraftKings official MLB rules (draftkings.com/help/rules/mlb)

---

## Hitters

| Stat                   | Points  |
|------------------------|---------|
| Single                 | +3.00   |
| Double                 | +5.00   |
| Triple                 | +8.00   |
| Home Run               | +10.00  |
| RBI                    | +2.00   |
| Run Scored             | +2.00   |
| Walk (BB)              | +2.00   |
| Hit By Pitch (HBP)     | +2.00   |
| Stolen Base            | +5.00   |
| Sacrifice Fly          | +1.25   |
| Sacrifice Hit (Bunt)   | +1.25   |

> Strikeouts, groundouts, flyouts, and all other outs carry **no point penalty** for hitters.

---

## Pitchers

| Stat                        | Points  |
|-----------------------------|---------|
| Inning Pitched (per out)    | +2.25   |
| Strikeout                   | +2.00   |
| Win                         | +4.00   |
| Complete Game               | +2.50   |
| Complete Game Shutout       | +2.50   |
| No-Hitter                   | +5.00   |
| Earned Run Allowed          | -2.00   |
| Hit Against                 | -0.60   |
| Walk Against (BB)           | -0.60   |
| Hit Batsman (HBP)           | -0.60   |

### Pitcher Notes

- **Innings Pitched** are awarded per full out recorded:
  - 1 out (1/3 IP) = +0.75 pts
  - 2 outs (2/3 IP) = +1.50 pts
  - 3 outs (1 full IP) = +2.25 pts
- **CG + Shutout + No-Hitter** all stack. A no-hit complete game shutout earns +10 pts in bonuses alone (+2.5 CG + +2.5 Shutout + +5 No-Hitter).
- **Unearned runs** do NOT penalize pitchers.

## Lineup Construction

| Position | Slots |
|----------|-------|
| P        | 2     |
| C / 1B   | 1     |
| 2B       | 1     |
| 3B       | 1     |
| SS       | 1     |
| OF       | 3     |
| UTIL     | 1     |
| **Total**| **10**|

- Salary cap: **$50,000**
- Players may be stacked from the same team.
- Pitchers score independently from hitters — two-pitcher lineup is standard.
